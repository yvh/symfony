LinkedIn Notifier
=================

Provides LinkedIn integration for Symfony Notifier.

DSN example
-----------

```
LINKEDIN_DSN=linkedin://ACCESS_TOKEN:USER_ID@default
```

where:
 - `ACCESS_TOKEN` is your LinkedIn access token
 - `USER_ID` is your LinkedIn user id

Mailjet Notifier
================

Provides [Mailjet](https://mailjet.com) SMS integration for Symfony Notifier.

DSN example
-----------

```
MAILJET_DSN=mailjet://FROM:AUTH_TOKEN@default
```

where:
 - `AUTH_TOKEN` is your Mailjet SMS auth token
 - `FROM` is an alphanumeric sender ID

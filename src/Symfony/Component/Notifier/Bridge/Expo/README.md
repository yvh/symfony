Expo Notifier
=============

Provides [Expo](https://docs.expo.dev/versions/latest/sdk/notifications/) integration for Symfony Notifier.

DSN example
-----------

```
EXPO_DSN=expo://TOKEN@default
```

where:
 - `TOKEN` is your Expo Access Token

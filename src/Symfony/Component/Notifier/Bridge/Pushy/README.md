Pushy Notifier
==============

Provides [Pushy](https://pushy.me/) integration for Symfony Notifier.

DSN example
-----------

```
PUSHY_DSN=pushy://API_KEY@default
```

where:

 - `API_KEY` is your application's API key, viewable in Pushy dashboard

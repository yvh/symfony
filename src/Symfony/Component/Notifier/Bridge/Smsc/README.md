SMSC Notifier
=============

Provides [SMSC](https://smsc.ru/) integration for Symfony Notifier.

DSN example
-----------

```
SMSC_DSN=smsc://LOGIN:PASSWORD@default?from=FROM
```

where:
 - `LOGIN` is your login
 - `PASSWORD` is your API password
 - `FROM` is your sender (NB: text identity, not a phone number)

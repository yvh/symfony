Resend Bridge
=============

Provides Resend integration for Symfony Mailer.

Configuration example:

```env
# SMTP
MAILER_DSN=resend+smtp://resend:API_KEY@default

# API
MAILER_DSN=resend+api://API_KEY@default
```

where:
 - `API_KEY` is your Resend API Key
